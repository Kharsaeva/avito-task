# Avito-task

Это тренировочный проект, созданный в рамках обучения в школе программирования

### <a href="https://avito-task.herokuapp.com/" target="_blank">открыть</a>

## Функционал проекта:

- получение картинок с сервера
- открытие модального окна при клике на картинку
- адаптивность
- роутинг
